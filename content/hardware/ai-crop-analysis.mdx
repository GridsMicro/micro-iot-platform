### Camera Setup

**Option 1: ESP32-CAM** (~‡∏ø150-200)
- Built-in camera
- WiFi connectivity
- Perfect for plant monitoring

**Option 2: Raspberry Pi + Camera** (~‡∏ø1,500-2,000)
- Higher resolution
- More processing power
- Can run AI models locally

### ESP32-CAM Example:

```cpp
#include <WiFi.h>
#include <HTTPClient.h>
#include "esp_camera.h"

void captureAndSend() {
  camera_fb_t *fb = esp_camera_fb_get();
  
  if (!fb) {
    Serial.println("Camera capture failed");
    return;
  }
  
  // Send to AI API
  HTTPClient http;
  http.begin("https://your-platform.com/api/analyze-plant");
  http.addHeader("Content-Type", "image/jpeg");
  
  int httpCode = http.POST(fb->buf, fb->len);
  
  if (httpCode == 200) {
    String response = http.getString();
    Serial.println("AI Response: " + response);
    // Parse JSON response
  }
  
  esp_camera_fb_return(fb);
  http.end();
}

void loop() {
  // Capture image every 6 hours
  captureAndSend();
  delay(6 * 60 * 60 * 1000);
}
```

## Machine Learning Models

### 1. **Yield Prediction Model**

**Input Features:**
- Temperature (avg, min, max)
- Humidity (avg)
- Soil moisture (avg)
- Light hours per day
- Fertilizer amount (NPK)
- Plant age (days)

**Model**: Random Forest Regressor

**Training Data Example:**
```python
import pandas as pd
from sklearn.ensemble import RandomForestRegressor

# Load historical data
data = pd.read_csv('farm_data.csv')

features = ['temp_avg', 'humidity_avg', 'soil_moisture', 
            'light_hours', 'fertilizer_npk', 'plant_age']
target = 'yield_kg'

X = data[features]
y = data[target]

# Train model
model = RandomForestRegressor(n_estimators=100)
model.fit(X, y)

# Predict
new_data = [[28, 65, 45, 8, 50, 60]]  # Current conditions
predicted_yield = model.predict(new_data)
print(f"Predicted yield: {predicted_yield[0]:.2f} kg")
```

### 2. **Disease Classification Model**

**Using TensorFlow.js** (runs in browser):

```javascript
// In Next.js app
import * as tf from '@tensorflow/tfjs';

const loadModel = async () => {
  const model = await tf.loadLayersModel('/models/plant-disease/model.json');
  return model;
};

const classifyDisease = async (imageElement) => {
  const model = await loadModel();
  
  // Preprocess image
  const tensor = tf.browser.fromPixels(imageElement)
    .resizeBilinear([224, 224])
    .expandDims()
    .toFloat()
    .div(255.0);
  
  // Predict
  const predictions = await model.predict(tensor).data();
  
  const classes = [
    'Healthy',
    'Powdery Mildew',
    'Leaf Blight',
    'Aphids',
    'Leaf Spot'
  ];
  
  const maxIndex = predictions.indexOf(Math.max(...predictions));
  
  return {
    disease: classes[maxIndex],
    confidence: predictions[maxIndex]
  };
};
```

### Natural Language Rules

**Instead of coding, farmers can write:**

```
"If temperature > 30¬∞C for 2 hours, turn ON misting system"
"If soil moisture < 30%, water for 15 minutes"
"If disease detected with confidence > 90%, send LINE alert"
```

**AI converts to executable code:**

```javascript
const rules = [
  {
    condition: (data) => data.temperature > 30,
    duration: 2 * 60 * 60 * 1000, // 2 hours
    action: () => controlDevice('misting_system', 'ON')
  },
  {
    condition: (data) => data.soil_moisture < 30,
    action: () => irrigate(15) // 15 minutes
  }
];

// Rule engine
const checkRules = (sensorData) => {
  rules.forEach(rule => {
    if (rule.condition(sensorData)) {
      rule.action();
    }
  });
};
```
